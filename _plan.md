# Plan - Site Réparation Premium "Néon Urbain"

## Objectif principal
Créer un site visuellement IDENTIQUE à https://clikclak.ch/ dans sa STRUCTURE et son LAYOUT, mais avec une identité visuelle moderne, premium et unique "néon urbain" dark mode.

## Stack technique
- Vue.js 3 + Composition API + `<script setup>`
- Vite
- Tailwind CSS avec palette néon urbain
- Vue Router 4
- Lucide Vue Next
- Google Fonts: "Space Grotesk" + "Inter"

## Étapes de développement

- [ ] 1. Setup projet Vite + Vue 3 + Tailwind avec config couleurs néon urbain
- [ ] 2. Créer les composants Layout (TopBar, Navbar, Footer) avec design dark premium
- [ ] 3. Créer les composants UI réutilisables (boutons dégradé, cards glow, etc.)
- [ ] 4. Développer la page Accueil section par section (Hero, Services, About, etc.)
- [ ] 5. Créer les données des modèles iPhone/Samsung avec tarifs
- [ ] 6. Page Smartphones avec grid des marques
- [ ] 7. Page iPhone avec tabs verticaux et tableau prix
- [ ] 8. Dupliquer pour Samsung, Huawei, Xiaomi, Google Pixel, OPPO
- [ ] 9. Pages Contact, À propos, Ordinateurs, Tablettes
- [ ] 10. Animations, responsive design et polish final

## Design "Néon Urbain"
- Dark mode premium (fonds #0A0A0F, #12121A, #1A1A25)
- Accents néon: violet (#A855F7), rose (#EC4899), bleu (#3B82F6)
- Dégradés pour boutons et éléments clés
- Effets glow et bordures lumineuses
- Typography: Space Grotesk (titres) + Inter (corps)
